Steps on how to make a github page

Step 1: Research on google on how to make a website from github pages

Step 2: Try the method stated in the guide

Step 3: Realize that it just simply takes you to the read me file

Step 4: Regret Life

Step 5: Reread another guide and learn you have to use Github Actions

Step 6: Research how to make YAML

Step 7: Break your repository

Step 8: Learn how to properly make yaml and push it to the drive

Step 9: CELEBRATE

Key Takeaways:
  1. You need a lot of CSS to make a simple website look good
  2. Creating github page is kind of hard
  3. Learning how to properly align your drafts to your vision is the hardest part of webdev
  4. Drafting is everything
  5. Take your time and have fun
